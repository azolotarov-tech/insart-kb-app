# Use Cases

Delivery case studies organised by FinTech domain.

| Domain | Description |
|---|---|
| [Payments](payments/) | Wallets, transfers, POS, payment gateways |
| [Banking](banking/) | Digital banking, neobanking, core modernisation |
| [Lending](lending/) | Loan origination, credit scoring, BNPL |
| [Wealth Management](wealth-management/) | Investment platforms, robo-advisors, trading |
| [Insurance](insurance/) | Underwriting engines, claims, policy management |
| [RegTech](regtech/) | KYC/AML, compliance automation, reporting |
| [Accounting](accounting/) | Bookkeeping automation, invoicing, financial reporting |
| [Blockchain](blockchain/) | Crypto platforms, DeFi, tokenisation |
| [B2B Infrastructure](b2b-infrastructure/) | BaaS, embedded finance, FinTech APIs |

To add a new case, copy `_template.md` into the relevant domain folder and fill it in.
